How to use RoundAway

RSA Keys

Your key pair lives in keys/ in the project folder. Never share private.pem.

  • Drag & drop a .pem file onto a key panel
  • Or paste PEM text directly and save
  • On first launch you will be prompted to generate keys

File Encryption

  1. Go to File > Encryption
  2. Drop any file onto the zone
  3. Set a password and algorithm
  4. Click Encrypt — saves as .enc

File Decryption

  1. Go to File > Decryption
  2. Drop the .enc file
  3. Enter the password — algorithm auto-detected
  4. Click Decrypt

Text Encryption

  1. Go to Text > Encryption
  2. Enter password and choose algorithm
  3. Paste plaintext and click Encrypt
  4. Copy the Base64 output packet

Text Decryption

  1. Go to Text > Decryption
  2. Paste the Base64 encrypted packet
  3. Enter the same password and click Decrypt

Use Send to Decrypt on the Encryption page to transfer output automatically.

Algorithms

  • AES-256-GCM — recommended, authenticated
  • AES-192 / 128-GCM — lighter GCM variants
  • AES-256 / 192 / 128-CBC — classic, no auth
  • CAMELLIA-256 — ISO/IEC 18033 standard
  • ChaCha20-Poly1305 — TLS 1.3 stream cipher

All keys derived via PBKDF2-SHA512, 200k iterations, random salt.
